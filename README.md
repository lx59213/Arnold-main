# Arnold - 合成大尖尖

一个魔性爆表的合成游戏，每次合成都会触发阿诺经典语音，灵感来自2048，但比2048更上头！

## 游戏截图
![游戏截图](样例.png)

## 游戏特色
- 🔊 **魔性语音**：每次合成都会触发阿诺的经典语音片段，让你根本停不下来
- 🎮 **玩梗合成**：不是普通的数字，而是充满梗的文字组合
- 📊 **分数系统**：努力刷新最高分（当前：3352），看看你能不能超越自己
- 🎯 **上头玩法**：开始容易上瘾，建议备好零食...

## 玩法说明
键盘上下左右随便按，相同的方块自动合成，然后...
就会听到那魔性的声音 🔊 

## 开始玩
1. 下载完整游戏包
2. 点击 `index.htm` 或拖进浏览器
3. 准备好被洗脑！
